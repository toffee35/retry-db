# Security Policy

## Supported Versions

As this is a pet project developed in my free time, security support is only 
provided for the **latest version**. Please ensure you are always using the 
most recent release.

## Reporting a Vulnerability

I take the security of this project seriously. If you discover a security 
vulnerability, I would greatly appreciate your help in disclosing it to me 
privately.

**Please do NOT open a public GitHub issue.**

Instead, please use GitHub's private vulnerability reporting feature. This 
will ensure your finding is handled confidentially.

1.  Go to the "Security" tab of the repository.
2.  Click on "Report a vulnerability".
3.  Fill out the form with as much detail as possible.

I will do my best to review and address the issue as soon as my time allows. 
Thank you for helping keep this project secure!
